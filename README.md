Track Ranker Pro
A professional-grade, lightweight tool for audio submission management. Rank, star, and organize your music tracks across multiple lists with an intuitive drag-and-drop interface.


Key Features:
Local-First Processing: No server uploads; all audio is handled locally via the Web Audio API.

Analytical Waveforms: Automatically generates visual waveforms for every .mp3 or .wav file added.

Multi-List Management: Organize tracks into separate tabs with individual ranking orders.

Persistent State: Your progress, notes, and rankings are automatically saved to your browser's local storage.

Smart Relinking: Easily reconnect audio files to your lists using the "Relink Folder" feature.

Export Ready: Download your rankings as a formatted text file for easy sharing.


Keyboard Shortcuts:
Space: Play / Pause

Arrow Right: Seek forward 5 seconds

Arrow Left: Seek backward 5 seconds


Built With:
HTML5 / CSS3

Vanilla JavaScript

SortableJS: For fluid drag-and-drop ranking

Web Audio API: For real-time waveform rendering

Gemini: AI-assisted development and optimization


Quick Start:
Open the HTML file in any modern web browser.

Drag and drop your audio files anywhere onto the page.

Use the drag handle to rank your tracks.

Add notes, star your favorites, and export when finished.
